# Инструкция по написанию кода

## Основные правила

### 1. Краткость и лаконичность
- Пиши код максимально кратко и по существу
- Избегай избыточных комментариев в коде
- Используй понятные имена переменных вместо длинных пояснений
- Не добавляй лишние абстракции без необходимости

### 2. Документация
- **НЕ создавай** документацию автоматически
- **НЕ пиши** README файлы без явного запроса
- **НЕ добавляй** PHPDoc и docblock комментарии, если не попросили
- Пиши документацию **ТОЛЬКО** когда пользователь явно об этом попросит

### 3. Тестирование
- **НЕ пиши** тесты автоматически
- **НЕ создавай** тестовые файлы без запроса
- **НЕ запускай** тесты самостоятельно
- Создавай тесты **ТОЛЬКО** по явной просьбе пользователя

### 4. Уточняющие вопросы
- **ВСЕГДА задавай вопросы**, если требования неясны
- Уточняй технические детали до начала работы
- Спрашивай о предпочтениях, если есть несколько вариантов решения
- Не додумывай за пользователя — лучше переспроси

### 5. Структура ответа
- Сначала — код
- Кратко — что делает код (1-2 предложения)
- Без лишних объяснений, если не просят

## Примеры

### ❌ Плохо (избыточно)
```php
<?php
/**
 * Модуль для работы с пользователями.
 * Содержит функции для создания и валидации пользователей.
 */

/**
 * Создает нового пользователя.
 *
 * @param string $name  Имя пользователя
 * @param string $email Email пользователя
 * @return array<string, string> Данные пользователя
 * @throws InvalidArgumentException
 */
function createUser(string $name, string $email): array
{
    // Валидация имени
    if (empty($name)) {
        throw new InvalidArgumentException('Name is required');
    }

    // Валидация email
    if (!str_contains($email, '@')) {
        throw new InvalidArgumentException('Invalid email');
    }

    return ['name' => $name, 'email' => $email];
}

// Тесты
function testCreateUser(): void
{
    assert(createUser('John', 'john@mail.com') === ['name' => 'John', 'email' => 'john@mail.com']);
}
```

### ✅ Хорошо (лаконично)
```php
<?php
function createUser(string $name, string $email): array
{
    if (empty($name) || !str_contains($email, '@')) {
        throw new InvalidArgumentException('Invalid input');
    }
    return ['name' => $name, 'email' => $email];
}
```

## Что делать, если пользователь просит

| Запрос | Действие |
|--------|----------|
| "Добавь тесты" | Написать тесты |
| "Напиши документацию" | Создать документацию |
| "Протестируй" | Запустить тесты |
| Неясное требование | Задать уточняющие вопросы |

## Запомни
- Код должен быть рабочим и чистым
- Минимум комментариев — максимум ясности
- Тесты и документация — только по запросу
- Сомневаешься? Спроси!
