## Beauty Flow – отзывы и рейтинг из Яндекс.Карт

**Beauty Flow** – это небольшое веб‑приложение на Laravel + Inertia Vue, которое позволяет владельцу компании подключить страницу организации на Яндекс.Картах и просматривать её рейтинг и отзывы в удобном интерфейсе личного кабинета.

### Основные возможности

- **Подключение компании**: пользователь указывает ссылку на страницу организации в Яндекс.Картах.
- **Загрузка данных**: приложение извлекает ID организации из ссылки и подготавливает данные (рейтинг, количество отзывов, список отзывов).
- **Личный кабинет**: после авторизации пользователь видит страницу с общим рейтингом компании и списком отзывов.

### Технологии

- **Backend**: Laravel (PHP).
- **Frontend**: Inertia.js + Vue 3.
- **База данных**: SQLite (по умолчанию, можно заменить через `.env`).

Все чувствительные данные (ключи, пароли, токены и т.п.) должны задаваться только через `.env` и **не попадают в репозиторий**. В проекте используется файл `.env.example` как шаблон для локальной конфигурации.

### Структура и ключевые файлы

- **`routes/web.php`**: маршруты личного кабинета (`/settings`, `/reviews`, обновление отзывов), доступные только авторизованным пользователям.
- **`app/Http/Controllers/CompanyController.php`**: логика сохранения ссылки на Яндекс.Карты, загрузки/обновления данных компании и отдачи страниц `Settings` и `Reviews`.
- **`app/Services/YandexMapsService.php`**: сервис для извлечения ID организации из ссылки и возврата данных (сейчас используется мок‑ответ без реального API).
- **`app/Models/Company.php` и `app/Models/Review.php`**: модели компании и отзывов, связи компания → отзывы.
- **`database/migrations/*create_companies_table.php` и `*create_reviews_table.php`**: таблицы для хранения информации о компании и её отзывах.
- **`resources/js/Pages/Company/Settings.vue`**: страница настроек компании, форма ввода ссылки на Яндекс.Карты.
- **`resources/js/Pages/Company/Reviews.vue`**: страница с общим рейтингом и списком загруженных отзывов.
