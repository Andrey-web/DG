## Beauty Flow – отзывы и рейтинг из Яндекс.Карт

**Beauty Flow** – это небольшое веб‑приложение на Laravel + Inertia Vue, которое позволяет владельцу компании подключить страницу организации на Яндекс.Картах и просматривать её рейтинг и отзывы в удобном интерфейсе личного кабинета.

### Основные возможности

- **Подключение компании**: пользователь указывает ссылку на страницу организации в Яндекс.Картах.
- **Загрузка данных**: приложение извлекает ID организации из ссылки и подготавливает данные (рейтинг, количество отзывов, список отзывов).
- **Личный кабинет**: после авторизации пользователь видит страницу с общим рейтингом компании и списком отзывов.

### Технологии

- **Backend**: Laravel (PHP).
- **Frontend**: Inertia.js + Vue 3.
- **База данных**: SQLite (по умолчанию, можно заменить через `.env`).

Все чувствительные данные (ключи, пароли, токены и т.п.) должны задаваться только через `.env` и **не попадают в репозиторий**. В проекте используется файл `.env.example` как шаблон для локальной конфигурации.
